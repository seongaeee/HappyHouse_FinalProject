## 개요

-   개인 맞춤형 점수를 보여주는 부동산 매물 웹서비스
-   공공 데이터를 사용한 개인 맞춤형 점수 기능
-   Open API를 사용한 지도 기능 구현
-   회원 등록, 수정, 탈퇴
-   로그인, 로그아웃
-   QnA 게시판

## 개발환경

-   ### Server

    -   SpringBoot REST Server

-   ### Database

    -   MySQL
    -   MyBatis

-   ### Client
    -   Vue.js
    -   HTML 5
    -   Bootstrap 4
