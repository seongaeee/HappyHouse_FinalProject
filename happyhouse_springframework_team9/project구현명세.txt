9조 프로젝트 구현 명세 (임성애, 정물결)

0) Controller 기반 구조
 0-1. homeController : MVC 구조 / 로그인, 로그아웃, 아파트 매물 정보 제공
 0-2. AddressController : 지역 정보를 받아오기 위한 RestfulAPI.


1) 기본 기능
 1-1. 메인 페이지 : homeController에서 "/" 맵핑으로 메인 페이지 접속하도록 구현
 1-2. 동별 정보 검색 결과 페이지 : 
 	1-2-1. 메인페이지에서 ajax 비동기로 데이터 요청
 	1-2-2. 지역 데이터를 보내주는 REST API 구현 (AddressController)
 	1-2-3. 시-구/군-동 선택후 검색시 해당 지역 매물 정보 검색(searchPage.jsp)
 1-3. 아파트별 정보 검색 : 
 	동별 정보 검색 결과에서 아파트 이름으로 연결된 링크를 클릭하면
 	아파트별 상세 정보 확인 페이지로 이동(deepSearch.jsp)
 1-4. 회원정보 등록, 수정, 삭제 페이지
 	1-4-1. 로그인 이전 보이는 회원가입, 로그인 이후 보이는 회원정보 수정 Modal 구현
 	1-4-2. homeController에서 데이터 처리 후 main페이지로 이동,
 	1-4-3. 회원 가입, 수정시 세션에 정보 저장 후 자동 로그인
 1-5. 로그인 / 로그아웃 페이지
 	1-4-1. 로그인 이전 보이는 로그인, 로그인 이후 보이는 로그아웃  Modal 구현
 	1-4-2. homeController에서 데이터 처리 후 main페이지로 이동,
 	1-4-3. 로그인시 세션에 User 정보 저장 후, Nav 변경.
